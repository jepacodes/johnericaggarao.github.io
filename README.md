# Personal Profile Website

A minimalist, academic, and professional personal profile website built with React and Tailwind CSS.

## Features

- Clean, minimalist design with teal accent color (#005F73)
- Responsive layout for all devices
- Subtle grid background pattern
- Featured works section with manual curation
- LinkedIn integration

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment to GitHub Pages

```bash
# Build and deploy
npm run deploy
```

The site will be available at: `https://jepaisme.github.io/Personal-Github-Website`

## Customization

To add or modify featured works, edit the `featuredWorks` array in `src/components/DedicationSection.jsx`.
