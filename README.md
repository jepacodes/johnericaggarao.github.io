# Personal Profile Website
Welcome to my personal website!

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment to GitHub Pages

```bash
# Build and deploy
npm run deploy
```

The site will be available at: `https://tinyurl.com/johnericaggarao`

## Customization

To add or modify featured works, edit the `featuredWorks` array in `src/components/DedicationSection.jsx`.
